# Recruitment

## 1. Chuẩn bị
PHP + Composer + MySQL + NodeJs

## 2. Backend
* Khởi động MySQL và tạo bảng `recruitment`
* Tạo tệp `.env` từ `.env.example`
* `composer install`
* `php artisan migrate:fresh --seed`
* `php artisan serve`

## 3. Frontend
* Tạo tệp `.env` từ `.env.example`
* `npm install`
* `npm run dev`
